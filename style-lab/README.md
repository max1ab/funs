# Style Lab

Fun projects UI 风格探索和存档。

每种风格一个 CSS 变量文件 + docs 文档。

## 已有风格

- **文心 (Wenxin)** — 人文简约，温暖克制
  - CSS: `design-tokens-wenxin.css`
  - 文档: `docs/wenxin.md`

## 使用

```html
<!-- 引用某风格 -->
<link rel="stylesheet" href="../style-lab/design-tokens-wenxin.css">
<!-- 或在项目内复制一份 -->
```

## 添加新风格

1. 新建 `design-tokens-xxx.css`
2. 在 `docs/xxx.md` 写设计说明
3. 更新本 README
