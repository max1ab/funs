# UI 风格记录：文心 (Wenxin)

版本：v0.2  
记录时间：2026-05-17  
命名：**文心** — 人文简约，温暖克制。

---

## 设计理念

像一本干净的书或练字帖。不争抢注意力，让用户聚焦内容本身。

关键词：温暖、留白、克制、扫读、零装饰。

---

## 字体

- 中英文正文：`-apple-system, "PingFang SC", "Microsoft YaHei", "Noto Sans SC", sans-serif`
- 等宽/编码：`"SF Mono", "Consolas", "Courier New", monospace`
- 不加载外部 web font，使用系统原生字体

---

## 间距与圆角

- 间距基于 4px 网格：4, 6, 8, 12, 16, 20, 24px
- 圆角：6px (小) / 8px (中) / 16px (大) / 999px (全圆 pill)

---

## 布局

- 固定视口，`overflow: hidden`
- 三行网格：顶栏 (auto) / 主体 (1fr) / 底栏 (auto)
- 键盘图居中，max-width 860px
- 手机端单列，键盘图压缩

---

## 按钮

- 主按钮：暖黑底 `#2d261c` + 白字，圆角 8px
- 次要按钮：透明底 + 暖灰边框，hover 变琥珀色边框
- Pill 切换：全圆角，选中态暖黑底白字

---

## 交互反馈

- Toast 提示：底部居中浮层，2 秒自动消失
- 正确：绿色背景
- 错误：红色背景
- 键位高亮：琥珀色边框 + 浅底 + 微放大

---

## 适用场景

适合工具类、练习类、阅读类产品。不适用于强品牌、娱乐化、电商类场景。
